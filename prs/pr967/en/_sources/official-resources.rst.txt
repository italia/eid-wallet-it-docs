Official Resources
-------------------

This section contains all references useful for the design and implementation of the IT-Wallet System, including repositories, logos, graphic assets, and additional documentation.

Official Website
^^^^^^^^^^^^^^^^

The official website of the IT-Wallet System is available at the following address. If it is not immediately accessible, it will be made available as soon as possible:

- **IT-Wallet Official Website**: `https://www.wallet.gov.it <https://www.wallet.gov.it>`_

Graphic Assets
^^^^^^^^^^^^^^

The following official graphic assets are available in the `official_resources/ <https://github.com/italia/eidas-it-wallet-docs/tree/versione-corrente/official_resources>`_ folder of this repository and can be downloaded directly from GitHub:

- **IT-Wallet ID**: Official graphic element for identifying the IT-Wallet System - `Download IT-Wallet ID <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/IT-Wallet-ID.svg>`_
- **IT-Wallet Logo**: Official logo of the IT-Wallet System (SVG format) - `Download the logo <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/logoIT-Wallet.svg>`_
- **IT-Wallet Symbol**: Official pictogram of the IT-Wallet System - `Download the symbol <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/symbol-IT-Wallet.svg>`_
- **Trust Mark**: Official graphic element for attesting membership in the IT-Wallet System (ENG and ITA versions) - `Download Trust Mark ENG <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/trustmark-ENG.svg>`_, `Download Trust Mark ITA <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/it/images/svg/trustmark-ITA.svg>`_
- **Authentication Button**: Official button for authentication via IT-Wallet - `Download Authentication Button <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/Authentication-button.svg>`_

Repositories and Additional Resources
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This section contains assets, resources, and artifacts that cannot be distributed as technical documentation but must be provided in their native digital format. These include source files in formats such as SVG, raster images, code repositories, configuration files, and other digital assets that are essential for the design and implementation of IT-Wallet System Technical Solutions.

Any additional technical resources, repositories, and documentation will be made available in this section in future releases of these Technical Specifications.

Brand Manual
^^^^^^^^^^^^

The Brand Manual containing complete guidelines for the use of graphic assets and the visual identity of the IT-Wallet System will soon be available on the :ref:`official website <official-resources:Official Website>` and in the Official Resources section of these Technical Specifications.
