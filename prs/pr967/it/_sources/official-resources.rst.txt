Risorse Ufficiali
-----------------

Questa sezione contiene tutti i riferimenti utili per la progettazione e l'implementazione del Sistema IT-Wallet, inclusi repository, loghi, risorse grafiche e documentazione aggiuntiva.

Sito Ufficiale
^^^^^^^^^^^^^^

Il sito ufficiale del Sistema IT-Wallet è disponibile all'indirizzo seguente. Qualora non fosse immediatamente accessibile, sarà reso disponibile quanto prima:

- **Sito Ufficiale IT-Wallet**: `https://www.wallet.gov.it <https://www.wallet.gov.it>`_

Risorse Grafiche
^^^^^^^^^^^^^^^^

Le seguenti risorse grafiche ufficiali sono disponibili nella cartella `official_resources/ <https://github.com/italia/eidas-it-wallet-docs/tree/versione-corrente/official_resources>`_ di questo repository e possono essere scaricate direttamente da GitHub:

- **IT-Wallet ID**: Elemento grafico ufficiale per l'identificazione del Sistema IT-Wallet - `Scarica IT-Wallet ID <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/it/images/svg/IT-Wallet-ID.svg>`_
- **Logo IT-Wallet**: Logo ufficiale del Sistema IT-Wallet (formato SVG) - `Scarica il logo <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/logoIT-Wallet.svg>`_
- **Simbolo IT-Wallet**: Pittogramma ufficiale del Sistema IT-Wallet - `Scarica il simbolo <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/it/images/svg/symbol-IT-Wallet.svg>`_
- **Trust Mark**: Elemento grafico ufficiale per l'attestazione dell'appartenenza al Sistema IT-Wallet (versioni ENG e ITA) - `Scarica Trust Mark ENG <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/en/images/svg/trustmark-ENG.svg>`_, `Scarica Trust Mark ITA <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/it/images/svg/trustmark-ITA.svg>`_
- **Authentication Button**: Pulsante ufficiale per l'autenticazione tramite IT-Wallet - `Scarica Authentication Button <https://raw.githubusercontent.com/italia/eidas-it-wallet-docs/versione-corrente/docs/it/images/svg/Authentication-button.svg>`_

Componenti HTML
^^^^^^^^^^^^^^^^

I seguenti componenti HTML ufficiali sono disponibili nella cartella `official_resources/ <https://github.com/italia/eidas-it-wallet-docs/tree/versione-corrente/official_resources>`_ di questo repository:

Repository e Risorse Aggiuntive
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Questa sezione contiene asset, risorse e artefatti che non possono essere distribuiti sotto forma di documentazione tecnica ma devono essere forniti nel loro formato digitale nativo. Questi includono file sorgente in formati come SVG, immagini raster, repository di codice, file di configurazione e altri asset digitali essenziali per la progettazione e l'implementazione delle Soluzioni Tecniche del Sistema IT-Wallet.

Qualsiasi risorsa tecnica aggiuntiva, repository e documentazione aggiuntiva saranno rese disponibili in questa sezione nelle versioni successive delle presenti specifiche tecniche.

Brand Manual
^^^^^^^^^^^^

Il Brand Manual contenente le linee guida complete per l'utilizzo delle risorse grafiche e dell'identità visiva del Sistema IT-Wallet sarà prossimamente disponibile sul :ref:`sito ufficiale <official-resources:Sito Ufficiale>` e nella sezione Risorse Ufficiali di queste specifiche tecniche.
