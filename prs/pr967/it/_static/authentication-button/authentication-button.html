Authentication Button - IT-Wallet

Examples of usage of the official component for authentication via IT-Wallet.

Note: This component is based on the dev-kit-italia framework and uses the official visual identity of the IT-Wallet System.

Positioning Variants

The authentication button is available in three positioning variants for icon and label, according to the Brand Manual specifications.

1. Standard (Default)

Icon and label left-aligned. The button size depends on the label length.
